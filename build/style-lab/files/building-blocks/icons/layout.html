{{#markdown}} Icons are added by the `fonts.css` file with :before pseudo elements, so they will appear before whatever piece of text you associate them with. To make this association, you need to include particular class and data-icon attributes in the relevant HTML element: e.g. <h4><i class="icon icon-generic" data-icon="4"></i> Travel by plane</h4>. {{/markdown}}
