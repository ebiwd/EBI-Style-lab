``` There are optional `data` attributes: - `data-color`: set the background color; options: orange, grey, blue, green - `data-name`: set part of the tite (data-name + "is part of the ELIXIR infrastructure") - `data-description`: set the text in the small line of text - `data-more-information-link`: by default the banner will point to a page at www.elixir-europe.org. If you would prefer to link to a custom page explaining your relationship with ELXIR, add the full URL here - `data-use-basic-styles`: if you have not loaded the Foundation Framework CSS, set to `true` and will inject a few lines of css for basic formatting Get started: - [Try it on CodePen](https://codepen.io/khawkins98/pen/ZXdGQO?editors=1000) - [See an example implementation](https://www.ebi.ac.uk/style-lab/websites/sample-site/boilerplate/affiliates.html) {{/markdown}}
